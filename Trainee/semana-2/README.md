# Semana 2 - Fundamentos de TypeScript

Bem-vindo(a) à segunda semana do programa de trainee! 🎉

## Objetivo da Semana

Nesta semana, você vai dar os primeiros passos no mundo da programação. O foco é:

- Entender os **tipos primitivos** do TypeScript
- Aprender a usar **condicionais** para tomar decisões no código
- Trabalhar com **arrays** e **loops** para processar listas de dados
- Criar **interfaces** e **objetos** para modelar dados
- Dominar **funções** para organizar e reutilizar código

**Não se preocupe se parecer muito confuso no início!** Você vai aprender no seu próprio ritmo, e está tudo bem errar - faz parte do processo.

**Você NÃO precisa saber Git para começar!** Siga estes passos simples:

1. Acesse o repositório no GitHub: [https://github.com/UFMGInformaticaJr/Trainee-2025-02](https://github.com/UFMGInformaticaJr/Trainee-2025-02)

## O que você vai encontrar aqui

### 📄 `exercicios.ts`

Este é o arquivo principal dos exercícios. Ele está no formato **"preencha as lacunas"**, onde você vai:

1. Corrigir tipos `any` para os tipos corretos
2. Completar funções com a lógica necessária
3. Criar interfaces e objetos
4. Implementar condicionais e loops

### Estrutura do arquivo

O arquivo está dividido em **7 seções**:

1. **Tipos Primitivos** - Aprenda sobre `string`, `number` e `boolean`
2. **Operadores e Condicionais** - Use `if/else` para tomar decisões
3. **Arrays e Loops** - Processe listas de dados com `for`
4. **Interfaces e Objetos** - Modele dados estruturados
5. **Funções** - Organize seu código em blocos reutilizáveis
6. **Callbacks** - Funções como parâmetros (conceito avançado!)
7. **Desafios Extras** (Opcional) - Para quem quer ir além!

Cada seção tem:

- ✅ Comentários explicativos detalhados
- ✅ Dicas práticas
- ✅ Exemplos de uso
- ✅ Testes no final para validar seu código

## Como executar os exercícios

### Pré-requisitos

Você precisa ter instalado:

- **Node.js** (versão 16 ou superior)
- **npm** ou **yarn**

### Passo a passo

1. **Instale o TypeScript e o ts-node** (se ainda não tiver, já que foram instalados nos passos anteriores dessa semana):

```bash
npm install -g typescript ts-node
```

2. **Navegue até a pasta da semana 2**:

```bash
cd exercicios/semana-2
```

3. **Abra o arquivo `exercicios.ts`** no seu editor de código favorito (VS Code, por exemplo)

4. **Complete os exercícios** conforme as instruções nos comentários

5. **Execute o arquivo para testar**:

```bash
npx ts-node exercicios.ts
```

6. **Descomente os testes** gradualmente (remova as barras `//` no início das linhas de teste)

7. **Verifique se os resultados batem com o esperado**

### Dicas para executar

- Comece descomentando apenas os testes da Seção 1
- Complete os exercícios dessa seção
- Execute o arquivo e veja se os resultados estão corretos
- Só então passe para a próxima seção
- **Não tenha pressa!** É melhor ir devagar e entender bem do que correr e ficar confuso

## O que fazer se travar

Se você ficar preso em algum exercício:

1. **Releia os comentários** - as dicas estão lá por um motivo!
2. **Tente quebrar o problema** em partes menores
3. **Teste com valores simples** - às vezes ajuda usar números pequenos como 1, 2, 3
4. **Peça ajuda** - não tenha vergonha! Programar é uma atividade colaborativa
5. **Pesquise** - buscar no Google faz parte da rotina de todo desenvolvedor
6. **Respire e faça uma pausa** - às vezes a solução vem quando você menos espera
